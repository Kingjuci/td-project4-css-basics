Color Values

HEX Value

In CSS, a color can be specified using a hexadecimal value in the form;

#rrggbb

RGB Value

In CSS, a color can be specified using an RGB value using this formula;

rgb(red, green, blue)

HSL Value

In CSS, a color can be specified using hue, saturation, and lightness in the form;

hsl(hue, saturation, lightness)
